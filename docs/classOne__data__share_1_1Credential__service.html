Registers the specified endpoint with One Data Share using the specified credentials.

It is expected that the authentication token used to create this Credential_service object is valid and that a connection can be made to One Data Share. If these preconditions are not met, exceptions may be thrown.

Parameters
typethe endpoint type to register
cred_idborrowed reference to the credential identifier to associate with the registered endpoint
uriborrowed reference to the uri of the endpoint to register
usernameborrowed pointer to the username needed to log in to the endpoint or nullptr to register an endpoint without a username
secretborrowed pointer to the password needed to log in to the endpoint or nullptr to register an endpoint without a password
Exceptions
Connection_errorif unable to connect to One Data Share
