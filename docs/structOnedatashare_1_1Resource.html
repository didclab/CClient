Detailed Description

Represents a directory, file, or link, otherwise refered to as a resource, from an endpoint's file system.

std::optional<std::vector<Resource> > Onedatashare::Resource::contained_resources

List of the resources contained by this resource if this resource can contain other resources, no value otherwise.

A resource that containes no resources will have an empty list which distinguishes it from a resource that cannot contain resources.
